Sirius Control-System Web Server

Introduction

Static variables and parameters such as excitation curves of the magnets, IPs of beaglebone computers, and so on are stored in files and provided via http protocol in this wev server. These parameters are used accessed by EPICS IOCs, high level applications and scripts. The content of this page is located in a github repository lnls-sirius/control-system-constants.
